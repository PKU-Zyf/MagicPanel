# MagicPanel 1.0.0

面板数据集的优化与合并。 / A useful package to standardize, merge, and optimize panel data sets.

使用 `pip install MagicPanel` 命令安装本库；使用 `pip install --upgrade MagicPanel` 命令更新到最新版本。
